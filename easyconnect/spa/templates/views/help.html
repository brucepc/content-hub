How to use the Content Hub Lesson Planner

Home

Featured lessons and library content explained

Select Lesson Planner from the Content Hub home page.

Featured items give prominence to lesson and library content which is important to showcase to yourself or students once they land on the Lesson Planner home page. This gives the user a quick way to find what is important to them that day. By featuring what is relevant to students that week or day it will reduce the need to browse through the lessons or library pages and get straight into learning.

How many items can I feature on the home page?

You can feature an unlimited number of items on the home page but only 10 will display at a time. The user can navigate through the various pages if more than 10 items are featured.

Back to Top

Remove featured lessons and library content from the home page

You can use the yellow star icon within the lesson or content item box to remove them from the featured section on your home page. Once you click the icon the items will automatically remove itself from the page.

Lessons

What are lessons?

Lessons are collections of educational content that you create for your students from the content within your library. The lessons page is where you can view and manage the lessons you have created. On this page you can also create a new lesson, feature a lesson, delete one or multiple lessons and edit the lesson names.

You can feature lessons so they appear on the home page of both the teacher and student views. This allows you to give more prominence to lessons that are being currently taught.

Back to Top

How to create a lesson

  1. Navigate to the Lessons page
  2. Click the Create A New Lesson button on the top right of the content area
  3. Enter a name in the Lesson Name text field and then click Create. If you do not wish to proceed with creating the lesson you can click the Cancel button.

You will then see a pop up window indicating that the lesson was successfully created.

Lessons can be sorted by date or name.

Back to Top

How to add content to a lesson

Once a lesson has been created you can add content from your library to it.

  1. Navigate to the Lessons page
  2. Find the lesson from the listing that you want to add content to

  3. Click the plus icon under the lessons name. This will bring you to the library page
  4. Here you can drag and drop content from the listing into the lesson in the left column.

To add multiple library items into a lesson at the one time select the check box for each lesson library item and click the Add to Lesson button.

Once you click this the button will change to a dropdown menu where you select the lesson. Once you’ve selected the lesson click the tick button. If you want to cancel at any point click the X button

Back to Top

How to edit a lesson name

  1. Navigate to the Lessons page
  2. Find the lesson from the listing that you want to edit
  3. Click the pencil shaped icon and a text field will appear. You can edit the lesson name from here. To save your changes click the tick icon. To cancel the edit at any time just click the X button.

You can also edit a lesson name from the lesson detail page via the pencil icon beside the lesson title.

Back to Top

How to delete content items from a lesson

  1. Navigate to the Lessons page
  2. Find the lesson from the listing that you want to remove content from
  3. Click the lesson icon or name to go to the lesson detail page
  4. Use the checkbox to the right of the content items to select which ones you want to remove
  5. Click the Remove From Lesson button on the top right of the content area
Back to Top

How to feature a lesson

  1. Navigate to the Lessons page
  2. Find the lesson from the listing that you want to feature
  3. Click the star icon below the lesson name. Yellow stars indicate that the lesson is featured, grey stars mean the lesson is not currently featured.

Once a lesson has been featured it will be displayed on the home page of both the teacher and student view.

Back to Top

How to delete a lesson

  1. Navigate to the Lessons page
  2. Use the checkbox underneath the lesson name to select the lesson you want to delete
  3. Click the Delete button near the top right of the content area

You can also delete a lesson from the lesson detail page via the Delete This Lesson button near the top right of the content area.

Library

What is the Library?

The Library page displays the content that you have uploaded and any preloaded content that exists. On this page you can:

  • Browse items by categories and tags
  • Edit the information related to uploaded content
  • Add content to lessons
  • Delete uploaded content
  • Upload content
  • Feature content
  • Hide/show content to students
Back to Top

What is the difference between Preloaded and Uploaded content?

The application can include preloaded educational content, i.e. educational content loaded before it begins to be used by teacher and students. This content may be loaded by, for example, the equipment manufacturer or a Ministry of Education before it is shipped to the school or educational establishment in which it is being used. Preloaded content cannot be deleted or edited.

Uploaded content is content that you the teacher upload from files on your computer. These can be any type of file – Excel document, Word document, HTML, PDF, E-Pub, video, image, etc. Uploaded content can be deleted and edited.

How to manage content items

Feature content

  1. Navigate to the Library page
  2. Use the star icon to the right of the library item you want to feature. Yellow means the content is featured, grey means it is unfeatured.

This content will appear on the Home page under Featured Content.

Back to Top

Edit uploaded content details

  1. Navigate to the Library page
  2. Click the Uploaded Content tab
  3. Click the pencil icon beside the content item that you want to edit

You will be brought to a form with fields to edit the title, description, tags and categories related to that piece of content. Once you are done editing click Save

Back to Top

Delete your Uploaded content

  1. Navigate to the Library page
  2. Click the Uploaded Content tab
  3. Use the checkbox to the right of the library item to select the one(s) you want to delete
  4. Click the Delete button to the right of the My Uploaded Content tab

You cannot delete preloaded content, only uploaded content may be deleted.

Back to Top

Add content to a lesson

  1. Navigate to the Library page
  2. Use the checkbox to the right of the library item to select the one(s) you want to add to a lesson
  3. Use the Add To Lesson dropdown to the right of the My Uploaded Content tab to select the lesson you want to add to. Once you've done this click the tick button beside it

  4. Once the lesson is visible in the lesson box to the left, you can drag and drop content from the library directly into the lesson.

To add multiple library items into a lesson simultaneously select the check box for each lesson and click the Add to Lesson button.

Once you click this the button will change to a dropdown menu where you select the lesson. Once you’ve selected the lesson click the tick button. If you want to cancel at any point click the X button.

Library items can be filtered using the category tree view menu

There is a limit of three levels of categories. Any content item that isn’t categorised can be found in the ‘uncategorised’ category.

You can add preloaded and uploaded content to lessons.

Hide & Show individual content items

  1. Navigate to the Library page
  2. Use the eye icon to hide the content from a student. When the eye is crossed out the content is hidden from students

Hiding content from the students will hide it from their library view and also from any lesson it is in.

Uploading my own content

You can upload almost any file type – .jpg, .pdf, e-pub, Word document, Excel document, video, audio, html and more.

  1. Navigate to the Library page
  2. Click the Upload Content button
  3. Select File and browse to your file(s) on your computer.
  4. Multiple files can be uploaded simultaneously from your computer

  5. The title of the file(s) will be entered in to the title field automatically. This is the only mandatory field.
  6. Description, Category and Tags are also added here.
  7. Enter a description into the description field.
  8. Enter relevant tags into the Tags Field. To enter more than one simply click Enter after each word.
  9. Only one category can be selected.
  10. This metadata will be applied to all files uploaded at this point (excluding IMS packages).

  11. Once you've finished entering the information click Upload
  12. Files that are not categorised will appear in the uncategorised category.

How to apply filters

The library filters are a set of criteria that can be applied all together or individually to filter the result set.

You can use the search box for a text based filter.

You can filter by a Category.

You can filter by a Tag.

Once added, the applied criteria can be removed.

What is the tag cloud and how do I use it?

The tag cloud displays a shortened list of tags that have been assigned to content items. Tags are descriptive words used to make searching for content easier. Tags such as "science", "lab-exercise", "juniors" and "pdf" applied to content items will allow those content items to show up in searches for any of those terms.

The more often a tag is applied to content, the larger the tag will display in the tag cloud. The tag cloud is an easy way to browse content by tags and an easy way to see what type of tags are the most common in your content.

Teacher Settings

How to manage categories

Click on the Manage button at the top right of the page. Choose Categories from the dropdown menu.

Add New Primary Category

  1. Click the Add New Primary Category button
  2. Type a name for the category
  3. Click the tick button to save your new category

Add a second or third level category

  1. Click the + icon beside the category you wish to be a parent of your new sub category
  2. Type a name for the category
  3. Click the tick button to save your new category

Edit A Category Name

  1. Click the pencil icon beside the category name you wish to edit
  2. Type a new name for the category
  3. Click the tick button to save your new category

Delete A Category

  1. Click the bin icon beside the category name you wish to delete
  2. Confirm you wish to delete the category in the pop up window

Please note: you cannot delete a category if content is assigned to it or it has a child category.

Back to Top

How to manage tags

Click on the Manage button at the top right of the page. Choose Tags from the dropdown menu.

Edit A Tag

  1. Click the pencil icon beside the tag name you wish to edit (tags for preloaded content cannot be deleted)
  2. Type a new name for the tag
  3. Click the tick button to save your new tag

Delete A Tag

  1. Click the bin icon beside the tag name you wish to delete
  2. Confirm you wish to delete the tag in the pop up window

Please note: you cannot modify tags associated with preloaded content.

Back to Top

USB/SD Import

Click on the Manage button at the top right of the page. Choose Settings from the dropdown menu and then select USB/SD Import.

Import Files From USB

  1. Use the arrows on the left side to open the various folders on the USB device
  2. Click the checkbox to the left of the filename to select what you want to import
  3. Select the top level folder if you wish to import all files within that folder. (only files at the root of the folder will be selected, sub folders and files within them will not get added)
  4. Click the Import button to start importing the files to your Library

Notes

  • Multiples files can be imported into the My Uploaded Content library
  • The title of each file will be based on the file name, and will be renamed with an incremental file name if the file name already exists
  • Please note: Description, Category and Tags are also added here. This metadata will be applied to all files uploaded at this point (excluding IMS packages).
  • Metadata can be updated on individual files through the Edit Content Information feature within the Library.
  • Folders cannot be imported.
  • Once you've finished entering the information click Import.
