How to use the Content Hub Lesson Planner

Home

What is featured content?

Featured items give prominence to lesson and library content which is important to showcase to yourself as you land on the Lesson Planner home page. This gives you a quick way to find what is important that day.

What are lessons?

Lessons are collections of educational content that is grouped together to learn about a specific topic or lesson.

Library

What is the Library?

The Library page displays the content that your teacher has uploaded or has been preloaded onto the device.

I can't see my library

Teachers have the ability to hide the library from students. If you need access to the library and can't see it, ask your teacher to enable it.

Accessing content

You can access content items by clicking on the title. Depending on the file type and your browser configuration, the file will either open in a new tab or your browser will handle downloading the file.

Back to Top

How to apply filters

The library filters are a set of criteria that can be applied all together or individually to filter the result set.

You can use the search box for a text based filter.

You can filter by a Category.

You can filter by a Tag.

Once added, the applied criteria can be removed.

Back to Top

What is the tag cloud and how do I use it?

The tag cloud displays the most frequently used tags within the library content. The more often a tag is applied to content, the larger the tag will display in the tag cloud. The tag cloud is an easy way to browse content by tags applied to that content and an easy way to see what type of tags are the most common in your content.
