How to use the Content Hub Lesson Planner

Managing Content

Overview

The Manage Content screen is displayed by default when the Administrator logs into the Lesson Planner.

An Administrator can:

  • Add/remove preloaded content packages
  • Remove teacher content
  • Remove categories

What is preloaded content?

Preloaded content is educational content loaded into the Lesson Planner before it is used by the teacher and students. This content may be loaded by, for example, the equipment manufacturer or a Ministry of Education before it is shipped to the school or educational establishment in which it is being used. Preloaded content can only be uploaded/imported/removed by an Administrator.

What is teacher content?

  1. Content that a teacher adds to the library in the lesson planner. These can be any type of file e.g. excel document, word document, HTML, PDF, E-Pub, video, image, etc.
  2. Lessons that a teacher creates

What are categories?

Categories provide a mechanism for teachers/administrators to group/classify content within the library. A content category can be:

  1. Assigned within a preloaded content package and created when preloaded content is uploaded/imported to the Lesson Planner by an Administrator
  2. Assigned when teacher content is uploaded/imported to the Lesson Planner by a Teacher
  3. Configured by a teacher through the Manage Category screen, or configured by an administrator through the Manage Category screen when viewing the Lesson Planner in teacher mode

Adding preloaded content

There are two ways to add preloaded content to the Lesson Planner

  1. Upload content via the browser by selecting the “Upload Package” link from the “Preloaded Content” section
  2. Import content via USB by selecting the “USB/SD Import Package” link from the “Preloaded Content” section

For files > 2GB the USB import option must be used. This is due to browser/server restrictions which may apply for larger file sizes.

Upload package

To upload a preloaded content package via the browser:

  1. Go to the “Preloaded Content” section in the Manage Content screen
  2. Select the “Upload Package” link
  3. In the Upload Package screen, select the “Choose Files” button to select the preloaded content package(s) you want to upload. If a preloaded content package with the same name already exists on the device, an option will be displayed to overwrite the existing package. If the Administrator chooses not to overwrite a package then that content package will not be uploaded to the device or imported into the Lesson Planner
  4. A message will be displayed confirming that the preloaded content package has been added successfully

USB/SD Import package

To import a preloaded content package via USB:

  1. Connect a USB device to the Content Access Point
  2. Go to the “Preloaded Content” section in the Manage Content screen
  3. Select the “USB/SD Import Package” link
  4. In the USB/SD Import Package screen, select the preloaded content packages you want to upload. If a preloaded content package with the same name already exists on the device, an option will be displayed to overwrite the existing package. If the Administrator chooses not to overwrite a package then that content package will not be uploaded to the device or imported into the Lesson Planner
  5. A message will be displayed confirming that the preloaded content package has been added successfully

Removing preloaded content

An Administrator can remove all preloaded content packages or individual preloaded content packages.

To remove all preloaded content packages:

  1. Go to the “Preloaded Content” section in the Manage Content screen
  2. Select the “Preloaded Content” checkbox. If there is no preloaded content, this checkbox will be disabled
  3. Select the Delete link
  4. All preloaded content on the device will be deleted from the lesson planner and from the device

To remove individual preloaded content packages:

  1. Go to the “Preloaded Content” section in the Manage Content screen. A list of existing preloaded content packages on the device will be listed
  2. Select the checkbox beside the preloaded content package you want to delete
  3. Select the Delete link
  4. The selected preloaded content package will be deleted from the lesson planner and from the device

Removing teacher content

To remove teacher content:

  1. Go to the “Teacher Content” section in the Manage Content screen
  2. Select the “Teacher Content” checkbox. If there is no teacher content, this checkbox will be disabled
  3. Select the Delete link
  4. All teacher content will be deleted from the lesson planner and from the device, i.e. all teacher uploaded library content and all lessons

It is not possible to delete individual items of teacher content in Administrator mode. However, an Administrator can delete individual items of content when in Teacher mode.

Removing categories

To remove categories:

  1. Go to the “Categories” section in the Manage Content screen
  2. Select the “Categories” checkbox. If there are no empty categories, i.e. categories with no associated content, this checkbox will be disabled
  3. Select the Delete link

Categories will only be deleted if they do not contain content, i.e. it is not possible to delete a category via the Manage Content screen if content is classified as belonging to that category

It is not possible to delete individual categories in Administrator mode. However, an Administrator can delete individual categories when in Teacher mode.

Teacher mode

An Administrator can perform all the functions of a Teacher when in Teacher mode

To use the Lesson Planner in Teacher mode, select the “Teacher mode” menu option on the Manage Content screen

When in teacher mode, an Administrator can return to the Manage Content screen by selecting the “Administrator mode” menu option
