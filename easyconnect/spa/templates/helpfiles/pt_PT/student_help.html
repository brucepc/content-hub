Página inicial

O que é o conteúdo em destaque?

Os itens em destaque dão visibilidade a conteúdo de lições e da biblioteca que é importante para si quando acede à página inicial do Responsável por planear a lição. Dessa forma, pode encontrar rapidamente o que é importante para si nesse dia.

O que são as lições?

As lições são compilações de conteúdo pedagógico que é reunido para a aprendizagem de um determinado tópico ou lição.

O que é a Biblioteca?

Como aplicar filtros

Os filtros da biblioteca são um conjunto de critérios que podem ser aplicados todos juntos ou individualmente para filtrar o conjunto de resultados.

Pode utilizar a caixa de pesquisa para criar um filtro baseado em texto.

Depois de adicionados, os critérios aplicados podem ser removidos.

O que é a cloud de etiquetas e como é que a utilizo?

A cloud de etiquetas mostra as etiquetas utilizadas com maior frequência no conteúdo da biblioteca. Quanto maior for a frequência de utilização de uma etiqueta, maior será a etiqueta na cloud. A cloud de etiquetas é uma maneira fácil de localizar conteúdo pelas etiquetas aplicadas e de ver que tipo de etiquetas é mais comum no conteúdo.
