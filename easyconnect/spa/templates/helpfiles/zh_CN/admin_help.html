如何使用内容中心的课程计划

内容管理

概述

当管理员登录课程计划时，内容中心会默认显示内容管理。

管理员可以进行如下操作：

  • 增加/删除预安装内容包
  • 移除教师内容
  • 移除分类

预安装内容是什么？

预安装内容会预存在课程计划中，教师和学生都可以使用。这些内容可以有不同的单位进行预装，例如，工厂，教育机构。只有管理员对预安装内容有管理权限（上传、导入和移除）。

教师内容是什么？

  1. 教师添加的内容—教师可以添加内容到课程计划的内容库中。内容可以是不同格式的文件，例如，Excel, Word, HTML, PDF, E-Pub, 音频，图片等。
  2. 教师创建的课程

分类功能是什么？

通过分类功能，教师和管理员能够对内容进行分组或分类。分类包含以下功能：

  1. 管理员可以对预安装的内容分类，也可以在上传或导入内容时分类
  2. 教师在上传或导入内容时分类
  3. 教师可以在分类管理页面进行配置，管理员可以在教师模式下的分类管理页面对内容进行配置

添加预安装内容

有两种方法可以添加预安装内容到课程计划

  1. 通过浏览器，在“预安装内容”部分选择“上传内容包”，然后上传需要的内容
  2. 插入USB移动硬盘，在“预安装内容”部分，选择“USB 导入包”导入内容

对于大于2GB的文件，必须用USB导入的方式。这是由浏览器对上传文件大小的限制导致的。

上传内容包

通过浏览器上传预装内容包：

  1. 在内容管理页面，打开“预装内容”功能
  2. 选择“上传内容”
  3. 在上传内容包页面，点击“选择文件”按钮，选择想要上传的内容包。如果预装内容包命名与已经存在的内容重名，会显示一个“覆盖已存在的内容包”选项。 如果管理员选择不覆盖内容包，内容包将不上传。
  4. 如果预装内容成功上传，将会有确认信息出现。

导入内容包

通过USB导入功能导入预装内容包：

  1. 将USB设备插入教育路由器
  2. 进入内容管理页面的“预装内容”功能
  3. 选择“USB导入内容包”
  4. 在USB导入内容包页面，选择需要上传得内容包。如果预装内容包命名与已经存在的内容重名，会显示一个“覆盖已存在的内容包”选项。 如果管理员选择不覆盖内容包，内容包将不上传。
  5. 如果预装内容成功上传，将会有确认信息出现。

移除预装内容

管理员可以移除所有预安装的内容包，也可以移除单个的内容包。

移除所有内容包：

  1. 在内容管理页面，打开“预装内容”功能
  2. 选取“预装内容”选项。如果没有预装内容，此选项将没有功能
  3. 点击“删除”
  4. 所有预装内容都会被删除。

删除单个预装内容包：

  1. 在内容管理页面，打开“预装内容”功能，所有预装内容会以列表形式显示。
  2. 选择想要删除的内容包
  3. 点击“删除”
  4. 所有选择的内容包将被删除。
回到顶部

移除教师内容

移除教师内容：

  1. 在内容管理页面，打开“教师内容”功能
  2. 选择“教师内容”选项。如果没有教师内容，此选项将无功能。
  3. 点击“删除”
  4. 设备中的所有教师内容包将被删除 （例如，教师上传内容和课程

在管理员模式下，单个内容包不能被删除。然而，管理员可以进入教师模式对单个内容包进行删除。

回到顶部

移除分类

移除分类：

  1. 在内容管理页面，打开“分类”功能
  2. 选择“分类”选项，如果分类中没有相关内容，此功能将无效。
  3. 点击“删除”

只有空的分类才可以被删除。如果有内容归于某个分类，此分类将不能被删除。

在管理员模式下，单个分类不能被删除。然而，管理员可以进入教师模式对单个内容包进行删除。

回到顶部

教师模式

管理员可以进入教师模式并进行所有教师模式下的操作

在内容管理页面下，选择菜单中的“教师模式”，然后可以使用课程划功能

在教师模式下，管理员可以通过选择菜单中的“管理员模式”返回内容管理页面
