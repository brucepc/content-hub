Utilisation de l’Outil de planification de leçons du Centre de contenus

Gestion des contenus

Présentation

L’écran Gérer le contenu (Manage Content) s’affiche par défaut lorsque l’administrateur se connecte à l’Outil de planification de leçons (Lesson Planner).

Un administrateur peut :

  • ajouter ou supprimer des paquets de contenus préchargés ;
  • supprimer des contenus d’enseignant ;
  • supprimer des catégories.

Qu’est-ce qu’un contenu préchargé (preloaded content) ?

Ce contenu pédagogique est importé dans l’Outil de planification de leçons avant que les enseignants et les élèves ne commencent à utiliser le logiciel. Par exemple, le fabricant ou le Ministère de l’Éducation peut importer des contenus avant la livraison des équipements à l’école ou l’établissement d’enseignement. Seul un administrateur peut charger, importer et supprimer des contenus préchargés.

Qu’est-ce qu’un contenu d’enseignant (teacher content) ?

  1. Il s’agit du contenu qu’un enseignant ajoute à la bibliothèque de l’Outil de planification de leçons, quelle qu’en soit la forme : classeur Excel, document Word, HTML, PDF, E-Pub, vidéo, image, etc.
  2. Les leçons créées par un enseignant.

Qu’est-ce que les catégories (categories) ?

Les catégories permettent aux enseignants et aux administrateurs de classer ou de regrouper des contenus dans la bibliothèque. Ces catégories peuvent être :

  1. Préassignées à un paquet de contenus préchargés et créées par l’administrateur au moment de l’importation ou du chargement des contenus préchargés dans l’Outil de planification de leçons.
  2. Assignées par l’enseignant lorsqu’il charge ou importe du contenu dans l’Outil de planification de leçons.
  3. Configurées par un enseignant dans l’écran Gérer les catégories (Manage Category) ou par un administrateur dans ce même écran lorsqu’il affiche l’Outil de planification de leçons en Mode enseignant.

Ajout de contenus préchargés

Il existe deux façons d’ajouter des contenus préchargés dans l’Outil de planification de leçons :

  1. Dans votre navigateur, cliquez sur le lien Télécharger un paquet (Upload Package) dans la section Contenu préchargé (Preloaded Content) pour télécharger du contenu.
  2. Pour importer un contenu stocké sur un périphérique USB, cliquez sur le lien Importer un paquet par USB (USB/SD Import Package) dans la section Contenu préchargé (Preloaded Content).

Les fichiers dont la taille dépasse 2 Go doivent être transférés via le port USB. En effet, au-delà de cette taille, certains navigateurs ou serveurs imposent des restrictions aux fichiers.

Téléchargement d’un paquet (Upload package)

Pour télécharger un paquet de contenus préchargés via le navigateur :

  1. Rendez-vous dans la section Contenu préchargé (Preloaded Content) de l’écran Gérer le contenu (Manage Content).
  2. Cliquez sur le lien Télécharger un paquet (Upload Package).
  3. Dans l’écran Télécharger un paquet (Upload Package), cliquez sur le bouton Sélectionner des fichiers (Choose Files) pour importer le ou les paquets de contenus préchargés. Si un paquet de contenus préchargés du même nom existe déjà dans le système, un message vous invite à confirmer que vous voulez remplacer le fichier existant. Si l’administrateur choisit de ne pas remplacer un paquet existant, le paquet de contenus ne sera pas chargé dans le système ni importé dans l’Outil de planification de leçons.
  4. Un message s’affiche pour confirmer l’ajout du paquet de contenus préchargés.

Importer un paquet par USB (USB/SD Import Package)

Pour importer un paquet de contenus préchargés via le port USB :

  1. Connectez un périphérique USB au Point d’accès au contenu (Content Access Point).
  2. Rendez-vous dans la section Contenu préchargé (Preloaded Content) de l’écran Gérer le contenu (Manage Content).
  3. Cliquez sur le lien Importer un paquet par USB (USB/SD Import Package)
  4. Dans cet écran, sélectionnez les paquets de contenus préchargés à importer. Si un paquet de contenus préchargés du même nom existe déjà dans le système, un message vous invite à confirmer que vous voulez remplacer le fichier existant. Si l’administrateur choisit de ne pas remplacer un paquet existant, le paquet de contenus ne sera pas chargé dans le système ni importé dans l’Outil de planification de leçons.
  5. Un message s’affiche pour confirmer l’ajout du paquet de contenus préchargés.

Suppression de contenus préchargés

Un administrateur peut supprimer tous les paquets de contenus préchargés ou certains paquets de contenus préchargés.

Pour supprimer tous les paquets de contenus préchargés :

  1. Rendez-vous dans la section Contenu préchargé (Preloaded Content) de l’écran Gérer le contenu (Manage Content).
  2. Cochez la case Contenu préchargé (Preloaded Content). Cette case est désactivée s’il n’existe aucun contenu préchargé.
  3. Cliquez sur le lien Supprimer (Delete).
  4. Tous les contenus préchargés dans le système seront supprimés de l’Outil de planification de leçons et du système.

Pour supprimer des paquets de contenus préchargés donnés :

  1. Rendez-vous dans la section Contenu préchargé (Preloaded Content) de l’écran Gérer le contenu (Manage Content). La liste des paquets existants de contenus préchargés dans le système s’affiche.
  2. Cochez la case en regard du paquet de contenus préchargés à supprimer.
  3. Cliquez sur le lien Supprimer (Delete).
  4. Le paquet de contenus préchargés sélectionné sera supprimé de l’Outil de planification de leçons et du système.

Suppression de contenus d’enseignant

Pour supprimer des contenus d’enseignant :

  1. Rendez-vous dans la section Contenu d’enseignant (Teacher Content) de l’écran Gérer le contenu (Manage Content).
  2. Cochez la case Contenu d’enseignant (Teacher Content). Cette case est désactivée s’il n’existe aucun contenu d’enseignant.
  3. Cliquez sur le lien Supprimer (Delete).
  4. Tous les contenus de l’enseignant seront supprimés de l’Outil de planification de leçons et du système, notamment les contenus que l’enseignant a stockés dans la bibliothèque et toutes les leçons.

En Mode administrateur, il est impossible de supprimer des contenus particuliers d’un enseignant. Toutefois, s’il se connecte en Mode enseignant, l’administrateur peut supprimer certains contenus.

Suppression de catégories

Pour supprimer des catégories :

  1. Rendez-vous dans la section Catégories (Categories) de l’écran Gérer le contenu (Manage Content).
  2. Cochez la case Catégories (Categories). Si toutes les catégories existantes sont vides, c'est à dire qu’aucun contenu ne leur est associé, la case est désactivée.
  3. Cliquez sur le lien Supprimer (Delete).

Les catégories seront supprimées uniquement si elles ne contiennent aucun contenu, c.-à-d. qu’il est impossible de supprimer une catégorie dans l’écran Gérer le contenu (Manage Content) tant que des contenus appartiennent à cette catégorie.

En Mode administrateur, il est impossible de supprimer des catégories particulières. Toutefois, s’il se connecte en Mode enseignant, l’administrateur peut supprimer des catégories données.

Mode enseignant

Lorsqu’il est connecté en Mode enseignant, l’administrateur peut effectuer les mêmes actions qu’un enseignant.

Pour utiliser l’Outil de planification de leçons en Mode enseignant, sélectionnez l’option de menu Mode enseignant (Teacher Mode) dans l’écran Gérer le contenu (Manage Content).

Lorsqu’il est connecté en Mode enseignant, un administrateur peut revenir à ce même écran en sélectionnant l’option de menu Mode administrateur (Administrator Mode).
