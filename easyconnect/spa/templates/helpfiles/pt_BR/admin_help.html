Como usar o Planejamento de aulas do Hub de conteúdo

Gerenciando conteúdo

Visão geral

A tela Gerenciar conteúdo (Manage Content) é exibida automaticamente quando o Administrador acessa o Planejamento de aulas (Lesson Planner).

Um Administrador pode:

  • Adicionar/remover pacotes de conteúdo pré-carregados
  • Remover conteúdo para professores
  • Remover categorias

O que é conteúdo pré-carregado (preloaded content)?

Conteúdo pré-carregado é o conteúdo educativo carregado no Planejamento de aulas antes de ser utilizado por professores e alunos. Este conteúdo pode ser carregado, por exemplo, pelo fabricante do produto ou pelo Ministério da Educação antes de o equipamento ser enviado às escolas ou à instituição de ensino em que será utilizado. O Conteúdo pré-carregado só pode ser atualizado/importado/removido por um Administrador.

O que é conteúdo para professores (teacher content)?

  1. Trata-se do conteúdo que o professor adiciona à biblioteca no planejamento de aulas. Este conteúdo pode ser qualquer tipo de arquivo, por ex., documento em Excel, Word, HTML, PDF, E-Pub, vídeo, imagem, etc.
  2. Aulas criadas pelo professor.

O que são categorias (categories)?

As categorias fornecem um mecanismo para que os professores/administradores possam agrupar/classificar o conteúdo dentro da biblioteca. A categoria de um conteúdo pode ser:

  1. Atribuída dentro de um pacote de conteúdo pré-carregado e criada quando o conteúdo pré-carregado for atualizado/importado no Planejamento de aulas por um Administrador.
  2. Atribuída quando o conteúdo para professores é carregado/importado no Planejamento de aulas por um professor.
  3. Configurada pelo professor através da tela Gerenciar categoria ou por um Administrador através da tela Gerenciar categoria quando estiver visualizando o Planejamento de aulas no modo para professores.

Adicionando conteúdo pré-carregado

Há duas maneiras de adicionar conteúdo pré-carregado ao Planejamento de aulas:

  1. Carregar conteúdo através do navegador, selecionando o link “Carregar pacote” (Upload Package) da seção “Conteúdo pré-carregado” (Preloaded Content).
  2. Importar o conteúdo via USB, selecionando o link “Importar pacote via USB” (USB/SD Import Package) da seção “Conteúdo pré-carregado” (Preloaded Content).

Para arquivos maiores que 2GB deve ser utilizada a opção de importação de conteúdo via USB. Isto se deve às restrições do navegador/servidor aplicadas em arquivos de tamanho maior.

Carregar pacote (Upload package).

Para carregar um pacote de conteúdo pré-carregado através do navegador:

  1. Vá para a seção “Conteúdo pré-carregado” (Preloaded Content) na tela Gerenciar conteúdo (Manage Content).
  2. Selecione o link “Carregar pacote" (Upload Package).
  3. Na tela Carregar pacote, selecione o botão “Selecionar arquivos” (Choose Files) para selecionar o(s) pacote(s) de conteúdo pré-carregado que deseja carregar. Caso já exista no dispositivo um pacote de conteúdo pré-carregado com o mesmo nome, será exibida uma opção para substituir o pacote existente. Se o Administrador optar por não substituir o pacote, o pacote de conteúdo não será carregado no dispositivo ou importado no Planejamento de aulas.
  4. Será exibida uma mensagem confirmando que o pacote de conteúdo pré-carregado foi adicionado com sucesso.

Importação do pacote via USB (USB/SD Import package)

Para importar um pacote de conteúdo pré-carregado via USB:

  1. Conecte um dispositivo USB no Ponto de acesso ao conteúdo (Content Access Point).
  2. Vá para a seção “Conteúdo pré-carregado” (Preloaded Content) na tela Gerenciar conteúdo (Manage Content).
  3. Selecione o link “Importação do pacote via USB" (USB/SD Import Package).
  4. Na tela Importação do pacote via USB (USB/SD Import Package), selecione os pacotes de conteúdo pré-carregado que deseja carregar. Caso já exista no dispositivo um pacote de conteúdo pré-carregado com o mesmo nome, será exibida uma opção para substituir o pacote existente. Se o Administrador optar por não substituir o pacote, o pacote de conteúdo não será carregado no dispositivo ou importado no Planejamento de aulas.
  5. Será exibida uma mensagem confirmando que o pacote de conteúdo pré-carregado foi adicionado com sucesso.

Removendo conteúdo pré-carregado

O Administrador pode remover todos os pacotes de conteúdo pré-carregado ou individualmente.

Para remover todos os pacotes de conteúdo pré-carregado:

  1. Vá para a seção “Conteúdo pré-carregado” (Preloaded Content) na tela Gerenciar conteúdo (Manage Content).
  2. Selecione a caixa de seleção “Conteúdo pré-carregado” (Preloaded Content). Caso não tenha nenhum conteúdo pré-carregado, esta caixa de seleção estará inativa.
  3. Selecione o link Apagar (Delete).
  4. Todo conteúdo pré-carregado no dispositivo será apagado do Planejamento de aulas e do próprio dispositivo.

Para remover pacotes de conteúdo pré-carregado individualmente:

  1. Vá para a seção “Conteúdo pré-carregado” (Preloaded Content) na tela Gerenciar conteúdo (Manage Content). Uma lista dos pacotes de conteúdo pré-carregado existentes no dispositivo será exibida.
  2. Selecione a caixa de seleção ao lado do pacote de conteúdo pré-carregado que deseja apagar.
  3. Selecione o link Apagar (Delete).
  4. O conteúdo pré-carregado selecionado será apagado do Planejamento de aulas e do dispositivo.

Removendo conteúdo para professores

Para remover o conteúdo para professores:

  1. Vá para a seção “Conteúdo para professores” (Teacher Content) na tela Gerenciar conteúdo (Manage Content).
  2. Selecione a caixa de seleção “Conteúdo para professores” (Teacher Content). Caso não tenha nenhum conteúdo para professores, esta caixa de seleção estará inativa.
  3. Selecione o link Apagar (Delete).
  4. Todo o conteúdo para professores será apagado do Planejamento de aulas e do dispositivo, ou seja, todo o conteúdo carregado pelo professor na biblioteca e todas as aulas.

Não é possível apagar itens individuais do conteúdo para professores no modo Administrador. No entanto, o Administrador pode apagar itens individuais de conteúdo quando estiver no modo para professores.

Removendo categorias

Para remover categorias:

  1. Vá para a seção “Categorias” (Categories) na tela Gerenciar conteúdo (Manage Content).
  2. Selecione a caixa de seleção “Categorias” (Categories). Se não houver categorias vazias, ou seja, categorias que ainda não foram associadas a nenhum conteúdo, esta caixa de seleção estará inativa.
  3. Selecione o link Apagar (Delete).

As categorias só serão apagadas se não estiverem associadas à nenhum conteúdo, ou seja, não é possível apagar uma categoria através da tela Gerenciar conteúdo (Manage Content) se o conteúdo estiver associado à tal categoria.

Não é possível apagar categorias individuais no modo Administrador. No entanto, o Administrador pode apagar categorias individuais quando estiver no modo para professores.

Modo para professores

Quando estiver no modo para professores, o Administrador pode executar todas as funções de um professor.

Para usar o Planejamento de aulas no modo para professores, selecione a opção “Modo para professores” (Teacher mode) no menu da tela Gerenciar conteúdo (Manage Content).

Quando estiver no modo para professores, o Administrador pode retornar à tela Gerenciar conteúdo (Manage Content) selecionando a opção “Modo administrador” (Administrator mode) no menu.
